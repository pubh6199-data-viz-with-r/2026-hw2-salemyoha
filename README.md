# Homework 2 — Visual Channels and Data Viz Design

Welcome to Homework 2 for PUBH 6199: Visualizing Data with R!

This assignment builds on your data visualization skills by asking you to **analyze and communicate patterns in real-world climate disaster data** from NOAA’s Billion-Dollar Disasters dataset. You’ll use what you’ve learned about **visual channels** (e.g., position, color, size) and **design principles** to create an effective data graphic and reflect on your design choices.

## ✍️ Instructions

1. **Clone your personal homework repo** (from GitHub Classroom):

   ```bash
   git clone https://github.com/pubh6199-data-viz-with-r/hw2-YOUR-GITHUB-USERNAME.git
   cd hw2-YOUR-GITHUB-USERNAME
   ```

2. Open the project in RStudio (open the .Rproj file or use File > Open Project). **DO NOT DOUBLE CLICK `.qmd` file to open**

3. Edit `2-hw2.qmd`:

- Fill in your name and the current date in the YAML header.

- Complete each question by writing code and your answers.

- Click the Render button to generate 2-hw2.html.

4. Review your rendered file `2-hw2.html` to make sure everything looks good.

## 🚀 How to Submit

There is no submit button — your submission is complete when you push your work to GitHub.

After completing the assignment:

    ```bash
    # Stage and commit your work
    git add 2-hw2.qmd 2-hw2.html
    git commit -m "Final submission for HW2"

    # Push it to GitHub
    git push origin main
    ```
Double check on GitHub:

- Go to your repo in a browser
- Confirm the latest commit includes both files
- View the rendered 2-hw2.html if needed, you may need to download the file to view it.

⏰ Only the latest commit before the deadline will be graded (unless extensions apply). Commits after the deadline will be marked as late.
